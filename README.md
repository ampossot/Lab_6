Population Genetics & Mendelian Genetics Simulators

Interactive web-based simulators designed for undergraduate genetics and evolutionary biology laboratories.

Developed for teaching applications in:

* Mendelian genetics
* Hardy-Weinberg equilibrium
* Selection
* Genetic drift
* Population genetics
* Statistical hypothesis testing using χ² analysis

⸻

Included Simulators

1. Dataset Generator — Mendelian Genetics

Interactive generator for:

* Dihybrid crosses
* Test-crosses
* χ² tests
* Expected phenotypic ratios
* Graphical visualization
* PDF report export

Students:

* calculate expected values
* test null hypotheses
* interpret segregation patterns

⸻

2. Population Genetics Simulator

Interactive Hardy-Weinberg simulator.

Generates:

* populations in HW equilibrium
* populations NOT in HW equilibrium
* random populations

Students:

* calculate allele frequencies (p and q)
* calculate expected genotype frequencies
* run χ² tests
* determine whether populations fit HW expectations

⸻

3. Selection Simulator

Simulation of natural selection acting on genotype frequencies.

Includes:

* positive selection
* selection against genotypes
* balancing selection / heterozygote advantage

Students observe:

* changes in allele frequencies
* genotype shifts after selection
* effects of selection strength

⸻

4. Selection Across Generations

Multi-generation evolutionary simulator.

Features:

* simulate one generation at a time
* allele-frequency trajectories
* genotype-frequency trajectories
* adjustable selection strength
* PDF export

Students visualize:

* gradual evolutionary change
* fixation
* maintenance of polymorphism
* effects of selection over time

⸻

5. Genetic Drift Simulator

Simulation of stochastic evolutionary change.

Features:

* multiple replicate populations
* adjustable population size
* bottleneck events
* allele-frequency trajectories for p and q
* fixation/loss detection
* PDF export

Students visualize:

* stochasticity
* founder effects
* bottlenecks
* random allele-frequency fluctuations
